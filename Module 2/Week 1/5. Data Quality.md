# Data Quality
- Issue of data quality
  - Is the data error free?
  - Reasons to take care of data quality
    - 1. Provies actionable insights.
      - Poor quality data -> poor analysis -> poor decisions
    - 2. Quality assurance means needed for regulatory compliance.
      - error can violate regulations can lead to legal complications.
    - 3. Quality leads to better engagement and interaction with external entities.
      - if application to be used by other/ data being used by third party software, important to gain organization's trust as leader provider.
      - Scientific data need to be as accurate as possible to enable human engagement and interaction with data system.
