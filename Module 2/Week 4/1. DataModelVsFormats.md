# Data Model Vs Data Format
- Data format is a serialized representation of data.
- CSV - term between 2 commas is value of attribute.
- Common notion that it is a relation with each record as a tuple and ith value in csv = ith attribute. (True)
- Data format of a csv file does not mean relational as it can't be represented in various formats like graph.
- Data model is same but format is csv in one and other graph.
